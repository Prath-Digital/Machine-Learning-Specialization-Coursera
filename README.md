# ΏθΥγ Machine Learning Specialization ΏθγΑ

Welcome to the **Machine Learning Specialization**! This repository contains all labs, assignments, and resources from the Coursera Machine Learning Specialization by DeepLearning.AI and Andrew Ng. Whether you're a beginner or looking to refresh your ML knowledge, this repo is your one-stop destination! ΏθΝθ

---

## ΏθΠΗ Courses Overview

### ΏθΥα Course 1: Supervised Machine Learning
- **Week 1:** Python, Model Representation, Cost Function, Gradient Descent
- **Week 2:** Numpy, Multiple Variables, Feature Scaling, Polynomial Regression, Scikit-learn
- **Week 3:** Classification, Sigmoid, Decision Boundary, Logistic Loss, Overfitting

### ΏθΥΩ Course 2: Advanced Learning Algorithms
- **Week 1:** Neurons & Layers, Coffee Roasting (TF & Numpy)
- **Week 2:** Backpropagation, Derivatives, Multiclass, ReLU, Softmax
- **Week 3:** Model Evaluation, Bias & Variance, Utilities
- **Week 4:** Decision Trees, Tree Ensembles

### ΏθΥβ Course 3: Unsupervised Learning, Recommenders, Reinforcement Learning
- **Week 1:** K-Means, K-Means++, Hierarchical Clustering
- **Week 2:** Principal Component Analysis (PCA), Recommender Systems
- **Week 3:** Reinforcement Learning, Q-Learning, Policy Gradient Methods
